 Lyan 自拍杆 DLL 操作手册

 1. 你拿到的文件

这个发布目录只包含：

- `Lyan_CS2自拍杆.dll`
- `操作手册.md`
- `selfiestick_bind.cfg`

从本项目侧看，真正运行所需的唯一文件只有 `Lyan_CS2自拍杆.dll`。  
手册和 `selfiestick_bind.cfg` 只是说明文件，不是运行依赖。

2. 在使用此文件前你必须已经有：

- 可正常运行的 `HLAE`
- 可正常运行的 `CS2`

也就是说：从你发出去的文件里，你只需要 `Lyan_CS2自拍杆.dll`；从运行环境上，你还需要自己的 HLAE + CS2。

3. 如何加载 DLL

1. 把整个 `zh-CN` 文件夹复制到任意不容易改动的位置。
2. 启动 `HLAE` 并进入 `CS2`。
3. 在 HLAE 控制台执行：

```cfg
mirv_loadlibrary "<Lyan_CS2自拍杆.dll 的绝对路径>"
```

示例：

```cfg
mirv_loadlibrary "D:\tools\selfiestick\zh-CN\Lyan_CS2自拍杆.dll"
```

如果加载成功，控制台会看到类似：

```text
[selfiestick] 已就绪 - 按 Ins 打开面板，F8 开关，F9 锁定，F10 跟随
```

4. 运行时热键

- `Ins`：默认显示 / 隐藏控制窗口
- `F8`：启用 / 关闭自拍杆
- `F9`：锁定当前观战目标
- `F10`：回到跟随模式

注意：
- 控制面板现在是一个可拖动、可缩放的独立窗口。
- 你可以在面板里修改“面板开关键”，修改后立即生效，并会保留到下次重新加载 DLL。
- `F8 / F9 / F10` 只在**面板隐藏**时生效。
- 面板显示时，输入由面板接管。
- 枪械绑定视角在开火时会抑制枪口抖动，不再随着每一枪明显抖镜头。
- 当目标使用 `AWP` 开镜时，自拍杆视角会保持原来的镜头位置与视野，不再出现开镜放大。

 5. 面板怎么用

面板里的核心功能只有一屏：

- 启用 / 关闭自拍杆
- `R / B / U` 局部微调
- 左侧自拍 / 右侧自拍 / 前视
- 修改面板按键
- 跟随 / 锁定当前 / 清除锁定
- 实时状态区

当前版本的语义是：

- `R` = Right，向右微调
- `B` = Back，向后微调
- `U` = Up，向上微调

 6. 常见使用流程

1. 先进入观战视角
2. 按当前面板按键打开窗口（默认是 `Ins`）
3. 点击启用
4. 按需要调整 `R / B / U`
5. 用“左侧自拍 / 右侧自拍 / 向前看”切换朝向
6. 用“锁定当前 / 跟随 / 清除锁定”管理目标

7. 常见状态说明

如果状态区里出现失败原因，表示 DLL 没有走兜底，而是明确告诉你真实阻塞点。

常见情况包括：

- 没有观战目标
- 锚点 attachment 缺失
- 目标解析失败
- 相机方向退化

看状态区的 `原因` 即可。
