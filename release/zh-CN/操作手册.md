# CS2 自拍杆 DLL 操作手册

## 发布目录内容

中文发布目录包含：

- `Lyan_CS2自拍杆.dll`
- `操作手册.md`
- `selfiestick_bind.cfg`

实际运行只需要 DLL。手册和 cfg 是说明文件。

## 支持版本

当前发布包标记支持：

- HLAE：`2.190.2`
- Counter-Strike 2：Steam app `730`，buildid `23669931`

CS2 和 HLAE 经常更新。如果后续游戏 schema、实体布局或 `SetUpView` 变化，DLL 可能会主动阻塞，并在面板里显示原因。

## 加载 DLL

1. 把整个 `zh-CN` 文件夹复制到稳定位置。
2. 启动 HLAE `2.190.2`，进入 CS2，并打开 demo 观战。
3. 在 HLAE 控制台执行：

```cfg
mirv_loadlibrary "<Lyan_CS2自拍杆.dll 的绝对路径>"
```

示例：

```cfg
mirv_loadlibrary "D:\tools\selfiestick\zh-CN\Lyan_CS2自拍杆.dll"
```

加载成功后，按 `Ins` 打开 ImGui 面板。

## 热键

- `Ins`：默认显示 / 隐藏面板
- `F8`：启用 / 关闭自拍杆
- `F9`：锁定当前观战目标
- `F10`：目标模式回到跟随

面板开关键可以在面板里修改，并保存到 DLL 同目录的 `selfiestick.ini`。

## 人物自拍模式

在 `相机模式` 里选择 `人物自拍`，使用原来的人物自拍杆。

主要控制：

- `左侧自拍`：从目标左侧构图自拍
- `右侧自拍`：从目标右侧构图自拍
- `向前看`：前视模式
- `跟随`：每帧使用当前观战目标
- `锁定当前`：锁定当前目标探员
- `清除锁定`：清除目标锁定
- `R / B / U`：Right / Back / Up 局部微调

内置构图会先应用，`R / B / U` 是在内置构图基础上的额外微调。

## 道具自拍模式

在 `相机模式` 里选择 `道具自拍`，相机会跟随当前目标探员投出的飞行道具。

支持道具：

- 烟
- 火，包含 molotov / incendiary
- 雷
- 闪
- 诱饵

行为规则：

- 目标探员仍然由 `跟随 / 锁定当前` 决定。
- 只接受当前目标探员 pawn 或 controller 拥有 / 投出的飞行道具。
- 如果当前只有 1 个有效飞行道具，并且没有锁定道具，会自动锁定。
- 如果同时有多个有效飞行道具，不会自动抢锁，需要在 `最近道具` 中手动点 `锁定`。
- `清除道具` 会清除当前道具锁定。如果下一帧仍然只有 1 个有效道具，可以再次自动锁定。
- 只跟踪飞行 projectile 实体。道具爆开、落地结束或实体消失后，相机会停止覆写，并在运行状态里显示原因。

道具控制：

- `道具 X / Y / Z`：道具相机局部偏移
- `俯仰 / 偏航 / 翻滚`：角度微调
- `最近道具`：显示候选道具、是否飞行中、句柄、帧龄、类型和锁定按钮

道具相机使用平滑预测锚点，减少慢放和正常速度下的一跳一跳、上下抽搐和卡顿感。

## 状态与排错

运行状态区会显示：

- 启用状态
- 相机模式
- 朝向
- 面板按键
- 目标模式
- 锁定目标 / 当前目标
- 道具锁定与候选
- 最近覆写结果
- 失败原因

日志会写在加载 DLL 的同目录：

```text
selfiestick_trace.log
```

如果替换 DLL 后仍加载旧版本，请完整重启 HLAE 和 CS2。Windows 会把已加载 DLL 保留在进程里，单纯覆盖文件不一定会立即生效。

## 安全说明

本 DLL 只用于 HLAE demo / 影视工作流，不用于匹配、实战或任何反作弊保护服务器。使用 HLAE 或注入工具进入 VAC 服务器存在风险。
